Purpose of this website

This website uses AJAX to access JSON files from the server and uses jQuery to dynamically add the retrieved data to the web page.
The content of this website is to show information from different countries.

Technologies being implemented

jQuery

jQuey encapsulates the commonly used functions in JavaScript, provides simple JavaScript operations, and optimises HTML document operations, event processing, animation design and AJAX operations.
jQuery provides a good CSS selector and encapsulates a large number of commonly used DOM operations, making development easier.

AJAX

AJAX is "Asynchronous JavaScript and XML", the main purpose of AJAX is to partially exchange data between the client and the server.
Traditional web applications will return a new page when the user completes the operation to update the page. This approach wastes a lot of bandwidth, because most of the HTML code in the two pages is often the same. AJAX only send and retrieve necessary data from the server and use JavaScript on the client side to process the response from the server.
Therefore, the server response will become faster because less data is exchanged between the server and the browser.

JSON

JSON is "JavaScript Object Notation", its a lightweight data exchange format.
JSON is a good data interchange language because of its cleanliness and clear hierarchy. It is easy for people to read and write, and it is also easy for machines to parse and generate, and it can effectively improve the efficiency of network transmission.
